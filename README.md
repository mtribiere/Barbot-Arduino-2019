# Barbot-Arduino-2019
Repo pour le code Arduino du barbot 2019


# Protocole
Message à envoyer sur le port serie de l'Arduino (Baudrate 9600)  

```
Boutton "Play" : 
!P?

La pièce tombe dans le trou X : 
!RX?
```
